IOT 담액식 수경재배(Deep Flow Technique) 시스템

# 기능

- 조명 및 산소펌프를 정해진 시간에 가동
- 온습도 모니터링
- 웹 UI (가동 시간 설정 및 온습도 그래프 시각화)

## 향후 구현 계획 
- 양액 농도 모니터링 기능
- 타임랩스 기능

# 하드웨어 구성

- ESP32 cam
- DHT22 온습도 센서
- 릴레이 x 3 (active-high)
- 5V 충전기 (ESP32 전원 공급)

# 핀아웃

| GPIO | 용도 | 비고 |
|---|---|---|
| 13 | DHT22 데이터 | |
| 4  | 릴레이 1 - 조명 | 온보드 플래시 LED |
| 12 | 릴레이 2 - 산소펌프 | 부트 스트랩(LOW) |
| 3  | 릴레이 3 - 팬 | UART0 RX |
